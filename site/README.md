# RehaSport Public (Astro)

Minimalistisches Public-Frontend für RehaSport.

## Stack
- Astro + React-Inseln
- Firebase Firestore (Live-Daten im Client)

## Entwicklung
```bash
cd site
npm install
npm run dev
```

## Build
```bash
cd site
npm run build
```

Output liegt in `site/dist` und wird über Firebase Hosting ausgeliefert.
